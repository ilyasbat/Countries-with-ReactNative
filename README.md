#### Installation

```bash
git clone https://github.com/ilyasbat/Countries-with-ReactNative.git
cd Countries-with-ReactNative
npm run install
cd ios 
pod install
````
#### Open in your iOS simulator
```bash
cd ../
npm run ios
``
